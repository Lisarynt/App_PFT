## ğŸ“± Fitur Utama
Aplikasi ini dirancang dengan fitur esensial untuk manajemen keuangan harian:
* **Sistem Autentikasi**: Keamanan data terjamin melalui sistem login akun menggunakan *Firebase Authentication*.
* **Dashboard Interaktif**: Ringkasan saldo, total pemasukan, dan pengeluaran dalam satu tampilan.
* **Pencatatan Transaksi**: Input data pemasukan dan pengeluaran secara terstruktur.
* **Budget Planning**: Membantu pengguna menetapkan batas pengeluaran agar tetap hemat.
* **Laporan Periodik**: Melihat grafik atau riwayat keuangan dalam periode tertentu.
* **Animasi**: Memiliki design yang unik dengan tema pixel dan animasi yang seru.

## âš™ï¸ Cara Menjalankan Aplikasi

### A. Instalasi di HP Android (Cara Termudah)
1.  Buka **Link Download APK** di atas melalui HP Android kamu.
2.  Download file bernama `app-release.apk`.
3.  Buka file tersebut di HP, lalu klik **Install**.
4.  Jika muncul peringatan *"Install from unknown source"*, izinkan saja karena aplikasi ini dibuat sendiri dan belum masuk Play Store.
5.  Aplikasi siap digunakan secara mandiri di HP tanpa perlu laptop.

### B. Menjalankan di Lokal (Development Mode)
1.  **Clone Repo**: `git clone https://github.com/Lisarynt/App_PFT.git`
2.  **Install Packages**: `flutter pub get`
3.  **Setup Database**: Pastikan endpoint **Mock API** sudah terpasang di dalam service aplikasi.
4.  **Run**: `flutter run`
